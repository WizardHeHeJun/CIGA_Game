# Pitfalls — 持久化错误记忆

> 跨会话沉淀的坑。**lint 能查的**（命名、序列化字段等）只进 `.claude/hooks/csharp-lint/rules.json`，不写这里；**lint 查不出的**（架构、生命周期、Unity 特有行为）写这里，并在 PRP 的「已知陷阱」按需注入。

每条格式：

```
### <错误名称>
- **错误**：<具体错误代码 / 现象>
- **修复**：<正确写法>
- **说明**：<为什么>（可选）
- **规则**：<对应 rule 文件 #章节，或 pitfall>
```
